Truncation of Sound

Demonstrates 3 different truncations of a single Sound() source.

Drag the range markers around to set offset and truncation points. Note that the markers can't be dragged while the sound is playing.

Some preset positions have been selected initially. Though the first truncation may seem to be "gapless MP3 looping", it would not work under heavy load, hence it is not true gapless looping unlike MP3Gapless Test.

Comes with tutorial.
